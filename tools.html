Tools

  • DLR_ICF: GitHub Link - Analyzes distal-to-local ratio (DLR) and inter-chromosomal fraction (ICF).
  • HiCPlot: GitHub Link - Plot square and triangle heatmaps from Hi-C matrices and tracks from bigwig files.
  • PausingIndex: GitHub Link - Calculate Promoter Pausing Index from RNApolII ChIP-seq data using longest TSS and TES.
  • MutSeq: GitHub Link - Generates in silico genetic/point mutations based on given DNA sequences.
  • ExpreLev: GitHub Link - Obtains TPM/CPM/FPKM for genes or peak regions based on raw read count matrices.
  • FullMatrix: GitHub Link - Generates a full matrix for each chromosome.
